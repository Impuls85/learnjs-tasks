Преобразование типов

"" + 1 + 0  - выражение примет значение "10"
"" - 1 + 0  - выражение примет значение -1
true + false  - выражение примет значение 1
6 / "3"  - выражение примет значение 2
"2" * "3"  - выражение примет значение 6
4 + 5 + "px"  - выражение примет значение "9px"
"$" + 4 + 5  - выражение примет значение "$45"
"4" - 2  - выражение примет значение 2
"4px" - 2  - выражение примет значение NaN
"  -9  " + 5  - выражение примет значение "  -9  5"
"  -9  " - 5  - выражение примет значение -14
null + 1  - выражение примет значение 1
undefined + 1  - выражение примет значение NaN
" \t \n" - 2  - выражение примет значение -2
